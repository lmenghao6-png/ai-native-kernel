三步开始

下载，然后启动

不需要编译代码。推荐先在 VMware、VirtualBox、UTM 或 QEMU 中体验。

01

下载 ISO

点击页面顶部的下载按钮，获得完整的 AegisOS 系统镜像。

02

创建虚拟机

选择 Linux 64 位、2 GB 内存和 8 GB 以上磁盘，并挂载下载的 ISO。

03

启动系统

选择 Live 直接体验，或选择 Install AegisOS 安装到虚拟磁盘。

LIVE 用户名 aegis-live
LIVE 密码 aegisos

安装时系统会要求创建你自己的管理员账户。

系统能力

不是聊天窗口，
而是系统的一部分

AegisOS 的 Agent 和 Guardian 能读取系统状态、管理服务、安装软件包并执行命令。 每次 root 操作都会写入审计记录，并提供本地紧急停止开关。

01持续监控磁盘、内存、进程、日志和网络
02内置 Qwen2.5 0.5B 模型，无需 API 密钥
03提供 Agent 与 Guardian 双运行时
04支持签名更新、快照与回滚工具
开发者预览版

当前镜像不含图形桌面，已内置可离线运行的 0.5B 小模型。Agent 和 Guardian 拥有 root 权限，不要在存有重要资料、密码或生产业务的电脑上使用。
